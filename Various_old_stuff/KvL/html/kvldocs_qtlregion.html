Search Option: QTL region

This option allows you to retrieve all the genes in a chromosomal region. The user is requested to submit a valid chromosome number and lower and upper coordinates (in basepairs) on that chromosome. It is not possible to use genetic distances (cM), nor is it allowed to use shorthand for mega- or kilobases (e.g. 59Mb in stead of 59000000). Note that no out-of-bounds checking is applied, if you give an unrealistic value (e.g. 0, or 1000000000) the program will simply crash and not return anything.

This search option uses the EnsEMBL api, which means it retrieves the the data directly from the EnsEMBL database. The EnsEMBL database version used for Fansnip is not always the latest as the api is update only twice a year.

What to expect:

Let's assume you have entered a valid chromosome name and coordinates.

It may take some time for results to appear. Then you will see:

A table will then appear displaying chromosome, gene start, gene end, EnsEMBL gene id, and – if available – a trivial name. Finally, a summary and description will be given for each of the genes, for example:


There are 55 genes in chromosome 1, bases 56000000 to 59000000 in chicken.

The description of the chicken gene ENSGALG00000012662 (EnsEMBL name: NUAK1):
NUAK family SNF1-like kinase 1 (EC 2.7.11.1) (AMPK-related protein kinase 5). [Source:Uniprot/SWISSPROT;Acc:O60285]

You will also notice a button stating 'submit'. If you press this button, the gene(s) that were found can be process in the SNPsearch phase. Alternatively, you can make a selection of your genes and submit.


If you did not submit a valid name you will be notified about it and you can choose to start a new query by pressing 'new'.

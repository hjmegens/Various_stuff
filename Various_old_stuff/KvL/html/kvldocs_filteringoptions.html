FanSnIP enables filtering of lists of genes, i.e. only the genes will pass that meet certain criteria.


Three filtering options are available:

  1. Expression

  2. Gene Set Enrichment analysis

  3. A set of genes supplied and uploaded by the user


Expression

After choosing 'expression' and hitting 'submit', you will be asked to check one or more tissues. If multiple tissues are selected, you can choose how to combine these. The default is 'AND'. You will also be asked to submit the level of expression. The level of expression can be 10 transcripts per million (TPM) and up, 100 TPM and up, or 1000 an up.

The database used for the 'expression filtering option' is the same as for selecting genes from expression profiles. These expression profiles are derived from the UniGene EST profiles. Data used in Fansnip is mined from the UniGene databased an stored in a local database. That local database is update only twice a year, so your results may not be completely identical to the UniGene database.


Gene Set Enrichment analysis

After choosing 'gene set enrichment â€“ GO' you will be asked to check a p-value. This value is used as a threshold for the gene set enrichment (also called over representation) analysis. Gene set enrichment is based on the simple premise that if you draw genes randomly from the 'gene universe', you should get a random distribution of the GO terms associated with those genes. If you do not draw genes randomly from the 'gene universe', for instance because you are looking at co-expressed genes, you expect a non-random distribution of GO terms. In stead you expect that certain GO terms will appear more often than expected based on a random draw of genes, and hence these GO terms will be 'over represented'. Overrepresentation is calculated based on comparing expected and observed values using a Fisher exact test. The test uses GO term associations based on a human annotation set implemented in the BioConductor package, because human is the species for which the best GO annotations are available.

The rationale for Gene Set Enrichment as a filtering option is that you want to find a set of genes that have similar functions or act in the same biological pathways. A pubmed search, for instance, could yield many genes and some may lie in the same pathways. Genes in the same pathway may be better candidates, and knowing the pathways improves the choice of genes.

Gene List

After choosing 'gene list', you will be asked to upload a text file with gene names. The gene names should be EnsEMBL ids, which you may have obtained from a previous search. This allows you to combine searches, for instance you may want to retrieve only those genes highly expressed in muscle that are also found on a certain region of chromosome 1.
